# 🚀 Ejecutar AutoDealers Flutter - Instrucciones Rápidas

## Opción 1: Script Automático (Más Fácil)

Desde PowerShell en la raíz del proyecto:

```powershell
.\EJECUTAR_FLUTTER_WEB.ps1
```

O el script simple:

```powershell
.\EJECUTAR_AHORA_SIMPLE.ps1
```

## Opción 2: Comandos Manuales

```powershell
# 1. Ir al proyecto Flutter
cd autodealers_flutter

# 2. Instalar dependencias
flutter pub get

# 3. Ejecutar en Chrome
flutter run -d chrome
```

## ⚡ Comandos Útiles

```powershell
# Ver dispositivos disponibles
flutter devices

# Analizar código
flutter analyze

# Limpiar y reinstalar
flutter clean
flutter pub get
flutter run -d chrome
```

## 📱 La Aplicación Se Abrirá Automáticamente

- La aplicación se compilará (primera vez puede tardar 2-5 minutos)
- Se abrirá automáticamente en Chrome
- Verás la pantalla de login
- Presiona `q` en la terminal para detener
