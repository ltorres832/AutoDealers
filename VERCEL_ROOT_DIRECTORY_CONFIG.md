## 🚀 Pasos para Configurar:

1. Haz clic en cada link de arriba
2. En la página de Settings → General
3. Busca "Root Directory"
4. Haz clic en "Edit"
5. Escribe el Root Directory correspondiente (ej: `apps/public-web`)
6. Guarda los cambios
