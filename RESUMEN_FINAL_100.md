## 🚀 CÓMO USAR

### Desarrollo
```bash
cd autodealers_flutter
flutter pub get
flutter run
```

### Build y Deploy
```bash
# Build solamente
.\scripts\build-flutter-web.ps1

# Build y Deploy
.\scripts\build-flutter-web.ps1 -Deploy
```

### Build Manual
```bash
cd autodealers_flutter
flutter build web --release
cd ..
firebase deploy --only hosting:flutter-web
```
